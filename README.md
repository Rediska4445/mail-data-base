# Mail-Ru: Application for mail's operations

[![GitHub Release](https://img.shields.io/github/v/release/Rediska4445/mail-data-base)](https://github.com/Rediska4445/mail-data-base/releases/latest) 
[![official VLPK](https://img.shields.io/badge/education-vlpk-orange.svg)](https://polcol.ru/)

Mail-Ru is application for working with mail database.
This application using Microsoft Sql Client and he's writed on C#/Windows Forms platform.

---

### He's contains this bases:
* printing_house;
* mail
* newspaper

---

### He's implement this function:
* Search information from tables and table
* Add new item to table
* Update exist item in table
* Remove item

---

## Warning

This application can be started only with working MySQL server!
Application working with any database (have possible for write own connection string, or name of DB),
but it is not be tested on really remote server, cuz application working only on local sever.
